# Luan's Task Manager

A simple task manager built using JavaScript that allows users to manage daily tasks directly in the browser.

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Data persistence using localStorage

## Technologies Used
- HTML
- CSS
- JavaScript
